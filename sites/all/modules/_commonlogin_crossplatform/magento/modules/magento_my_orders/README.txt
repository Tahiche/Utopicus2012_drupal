Magento My Orders
-------------------------------------------------------------------------------

DESCRIPTION
-------------------------------------------------------------------------------

This modules provides information about Magento Customers's Orders in Drupal.
What it actually does is showing all user's (Customer's) Orders on a separate
`Orders` tab of User's profile page user/%uid/orders. It retrieves all order list 
by customer_id from remote Magento.
Also this modules provides displaying extended Order details for each Customers' Order
on the separete page user/%uid/order/%order_id.

This module adds two permissions:
 `view own orders` to access information about user's own orders.
 `view any orders` to access information about anyone's orders.
