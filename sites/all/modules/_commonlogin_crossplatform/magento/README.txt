Magento module
-------------------------------------------------------------------------------

This is a core module of the Drupal-Magento integration modules package.
This module provides basic functions for data manipulation.
It requires Magento API module to connect to the remote Magento backend. 

INSTALLATION:
-------------------------------------------------------------------------------

Install as usual, see http://drupal.org/node/70151 for further information.

A comprehensive installation howto is already on its way.
